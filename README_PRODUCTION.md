## 🔨 あなたがやるべきこと

### 📝 すぐにやること（1-2日）

#### 1. APIキーの取得
```bash
# Stripe（決済）
https://stripe.com
→ アカウント作成
→ APIキー取得（Test & Live）

# Resend（メール送信）
https://resend.com
→ アカウント作成
→ APIキー取得

# Twilio（SMS - オプション）
https://twilio.com
→ アカウント作成
→ 電話番号取得
→ APIキー取得
```

#### 2. 環境変数設定
`.env`ファイルに追加：
```env
# Stripe
STRIPE_SECRET_KEY=sk_test_...
VITE_STRIPE_PUBLISHABLE_KEY=pk_test_...

# Resend
RESEND_API_KEY=re_...

# Twilio (オプション)
TWILIO_ACCOUNT_SID=AC...
TWILIO_AUTH_TOKEN=...
```

#### 3. ドメイン設定
- ドメイン購入（例: on-carry.com）
- Supabaseでカスタムドメイン設定
- Resendでドメイン認証
